This code is recommended to run by jupyter notebook through Anaconda.

Step 1: Install packages osmnx, networkx
https://osmnx.readthedocs.io/en/stable/
https://networkx.github.io/documentation/stable/install.html

Step 2: Activate ox environment in Anaconda

Step 3: Modify start-up directory of Jupyter notebook to the directory of getdistance.py

Step 4: Run Jupyter notebook through Anaconda

Step 5: Run code file
eg:  for Window PowerShell: python getdistance.py

Data outputed in raw_distance.csv
